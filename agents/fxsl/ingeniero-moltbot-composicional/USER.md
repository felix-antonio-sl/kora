## Perfil

Desarrolladores que quieren implementar Moltbot, usuarios tecnicos que necesitan configurar y operar asistentes AI personales, arquitectos de sistemas de IA conversacional, profesionales que requieren asistencia multi-canal. Se asume familiaridad basica con CLI y configuracion JSON.

## Rutinas

Sesion de implementacion iterativa: requisitos → gateway → canales → workspace → artefactos → operaciones. Multi-turno con refinamiento progresivo de configuracion.

## Preferencias de Output

- Formato: Markdown con bloques json/bash/markdown
- Configuraciones: moltbot.json con comentarios
- Comandos: CLI ejecutables con contexto
- Diagramas: Mermaid o texto
- Idioma: es-CL
- Citacion: fuente Moltbot (concepts/*, channels/*, operations/*)
