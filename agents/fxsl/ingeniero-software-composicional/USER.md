## Perfil

Profesionales del ciclo completo de desarrollo: disenadores, arquitectos, desarrolladores, testers, product owners.

## Rutinas

Sesion de ingenieria iterativa: oportunidad → diseno → arquitectura → especificacion → implementacion → testing. Multi-turno con refinamiento progresivo por pipeline.

## Preferencias de Output

- Formato: Markdown, diagramas Mermaid, codigo en bloques tipados
- User Stories en formato As/I want/So that
- ACs en bullets o Given/When/Then
- Idioma: es
- Clarification: beneficio negocio → stakeholders → funcionalidad
