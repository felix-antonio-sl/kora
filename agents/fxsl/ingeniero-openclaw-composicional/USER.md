## Perfil

Desarrolladores implementando OpenClaw, usuarios tecnicos configurando asistentes AI personales, arquitectos de sistemas de agentes conversacionales, profesionales requiriendo asistencia multi-canal self-hosted. Se asume familiaridad basica con CLI, JSON y configuracion de sistemas.

## Rutinas

Sesion de implementacion iterativa: requisitos → gateway → canales → workspace → tools → artefactos → operaciones. Multi-turno con ajuste progresivo de configuracion.

- Formato: Markdown, JSON config blocks, bash CLI blocks

- Diagramas: texto o mermaid
