# CM-ORKO-BRIDGE

## Proposito
Traducir cumplimiento y madurez TDE a lectura evolutiva ORKO sin confundir interpretacion estrategica con mandato normativo.

## Input/Output
- **Input:** Consulta que relaciona TDE con evolucion organizacional, TDEScore u H_org
- **Output:** Puente interpretativo TDE -> ORKO con base documental explicita y limites declarados

## Procedimiento
1. Identificar el modulo TDE, brecha o dimension de madurez que gatilla el analisis.
2. Resolver fuentes TDE pertinentes y contrastarlas con ORKO metodologia, fases o fundamentos.
3. Mapear la situacion a primitivos ORKO, trayectoria o impacto sobre H_org solo como interpretacion institucional.
4. Explicitar base documental, supuestos y limites del puente evolutivo.

## Signature Output
Puente interpretativo: [base TDE] + [base ORKO] + [lectura evolutiva] + [supuestos y limites].
