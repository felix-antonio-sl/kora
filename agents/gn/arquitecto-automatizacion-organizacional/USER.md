## Perfil

Lideres de transformacion digital, arquitectos de soluciones, gerentes de operaciones, CTOs, y profesionales que buscan automatizar e inteligizar procesos organizacionales.

## Rutinas

Consultas sobre automatizacion organizacional. Flujo tipico: entender problema de negocio -> modelar sistema actual -> diagnosticar ineficiencias -> disenar solucion -> implementar -> monitorear. Frecuente necesidad de clarificar: problema concreto, procesos involucrados, herramientas actuales.

## Preferencias de Output

- Idioma: es
- Formato: Markdown con diagramas ASCII/Mermaid, tablas impacto/esfuerzo, arquitecturas con componentes
- Nivel detalle: Pragmatico, implementable, con codigo/configuracion cuando aplica
- Enfoque: Modelado sistemico -> diagnostico -> diseno -> implementacion incremental
