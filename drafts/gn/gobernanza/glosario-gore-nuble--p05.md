# Glosario Unificado GORE Ñuble - Parte 05

## SYS
- **ENT SYS ACCESO LECTURA:** Permiso de nivel básico que permite consultar datos sin capacidad de modificación.
- **ENT SYS ACCESO VPN:** Conexión segura remota a la red privada institucional mediante túnel cifrado.
- **ENT SYS ACTIVO AUDIOVISUAL:** Equipo de hardware capturador o reproductor de medios bajo gestión técnica.
- **ENT SYS ACTIVO MULTIMEDIA:** Contenido digital (video, audio, interactivos) usado en la gestión del GORE.
- **ENT SYS ACTO:** Evento o registro formal de una transacción dentro de los sistemas digitales.
- **ENT SYS ACTOR:** Entidad (humana o agente) que interactúa con la plataforma GORE_OS.
- **ENT SYS AGENDA:** Módulo de gestión del tiempo y compromisos para los usuarios del sistema.
- **ENT SYS ALERTA:** Notificación automática disparada por el incumplimiento de una regla o umbral.
- **ENT SYS ALERTA CUMPLIMIENTO:** Aviso sobre el vencimiento de plazos legales o administrativos obligatorios.
- **ENT SYS ALERTA CUSTODIA:** Notificación de retiro o ingreso de documentos de valor de la caja de seguridad.
- **ENT SYS ALERTA MORA:** Aviso de retraso en pagos o rendiciones por parte de terceros ejecutores.
- **ENT SYS ALERTA REPUTACIONAL:** Aviso ante menciones críticas o crisis de imagen detectadas en el entorno.
- **ENT SYS ALERTA SEGURIDAD:** Notificación por intentos de acceso no autorizados o fallas de integridad.
- **ENT SYS ALERTA UMBRAL:** Aviso cuando una métrica financiera o técnica supera el límite permitido.
- **ENT SYS ALERTA VENCIMIENTO:** Notificación programada previa al término de la vigencia de un contrato o hito.
- **ENT SYS ALGORITMO CLASIFICACION:** Lógica automatizada para categorizar documentos o casos según patrones.
- **ENT SYS ANTECEDENTE PROYECTO:** Vínculo digital a los documentos de sustento técnico de una iniciativa.
- **ENT SYS ALMACENAMIENTO SEGURO:** Repositorio de datos con altos estándares de cifrado y control de acceso.
- **ENT SYS ANALISIS CAUSA RAIZ:** Metodología diagnóstica para identificar el origen de fallas operativas.
- **ENT SYS ANOMALIA DATO:** Inconsistencia o valor fuera de rango detectado en el modelo de datos.
- **ENT SYS ARQUITECTURA SISTEMA:** Diseño estructural de los componentes y flujos de información de GORE_OS.
- **ENT SYS ATOMO:** Unidad mínima e indivisible de información dentro del modelo de datos KODA.
- **ENT SYS BANDEJA TAREAS:** Interfaz que muestra las acciones pendientes asignadas a un rol específico.
- **ENT SYS BASE CONOCIMIENTO:** Repositorio curado de información técnica y normativa para apoyo a la gestión.
- **ENT SYS BASE DATOS:** Motor de almacenamiento estructurado de la información operativa del GORE.
- **ENT SYS BASE DATOS PERSONALES:** Registro que contiene información sensible bajo protección de la Ley 19.628.
- **ENT SYS BITACORA OPERATIVA:** Registro secuencial de todas las acciones realizadas en un módulo del sistema.
- **ENT SYS BUSQUEDA:** Capacidad de recuperación de información mediante términos clave o filtros.
- **ENT SYS BUSQUEDA SIMILARIDAD:** Búsqueda avanzada basada en cercanía semántica entre vectores de datos.
- **ENT SYS CALENDARIO:** Representación temporal de eventos, hitos y vencimientos institucionales.
- **ENT SYS CALIDAD DATOS:** Grado de exactitud, completitud y consistencia de la información en el sistema.
- **ENT SYS CARPETA DIGITAL:** Contenedor virtual que agrupa documentos relacionados con un mismo asunto.
- **ENT SYS CATALOGO SERVICIOS:** Listado de prestaciones digitales disponibles para usuarios internos y externos.
- **ENT SYS CERTIFICADO DIGITAL:** Archivo que acredita la identidad de un usuario o servidor en transacciones.
- **ENT SYS CHECKLIST:** Lista de verificación sistematizada para asegurar el cumplimiento de pasos operativos.
- **ENT SYS CHECKLIST CIERRE:** Verificaciones finales para concluir administrativamente un proceso en el sistema.
- **ENT SYS CHECKLIST INGRESO:** Validación de completitud de datos al cargar nuevos registros.
- **ENT SYS CHECKLIST NORMATIVO:** Validación del cumplimiento de requisitos legales en una tramitación digital.
- **ENT SYS CHECKLIST VERIFICACION:** Pasos técnicos para asegurar la integridad de una carga de datos (ETL).
- **ENT SYS CODIGO BARRAS:** Representación gráfica de IDs para control de inventario o correspondencia.
- **ENT SYS CODIGO IMPLEMENTACION:** Script o programa que ejecuta una lógica específica dentro de GORE_OS.
- **ENT SYS CODIGO INVENTARIO:** Identificador único asignado a un activo físico registrado en el sistema.
- **ENT SYS CONVENIO:** Registro digital de los términos y condiciones de un acuerdo interinstitucional.
- **ENT SYS CONVERTIDOR FORMATOS:** Herramienta que transforma archivos entre distintos estándares (ej. PDF a Texto).
- **ENT SYS CUELLO BOTELLA:** Punto del proceso donde el flujo se detiene por falta de capacidad o recursos.
- **ENT SYS DASHBOARD:** Panel visual interactivo que resume indicadores críticos de gestión.
- **ENT SYS DASHBOARD EJECUTIVO:** Vista de alto nivel con KPIs estratégicos para la autoridad regional.
- **ENT SYS DECISION TECNICA:** Registro de la opción elegida ante un dilema de implementación del sistema.
- **ENT SYS DEDUPLICACION DATO:** Proceso de identificación y eliminación de registros repetidos en la base de datos.
- **ENT SYS DIAT:** Declaración Individual de Accidente del Trabajo (versión digital).
- **ENT SYS DIAT DIEP:** Módulo de gestión de declaraciones de accidentes y enfermedades profesionales.
- **ENT SYS DICCIONARIO DATOS:** Definición técnica de cada campo y tabla del modelo de datos institucional.
- **ENT SYS DISPOSITIVO BIOMETRICO:** Hardware usado para validar identidad mediante rasgos físicos (huella, rostro).
- **ENT SYS DOCUMENTO:** Unidad fundamental de información; puede ser físico o digital.
- **ENT SYS DOCUMENTO DIGITAL:** Archivo electrónico con firma y metadatos que tiene validez administrativa.
- **ENT SYS DOCUMENTO SOPORTE:** Anexo que respalda la información contenida en un acto administrativo.
- **ENT SYS DOCUMENTO TECNICO:** Especificaciones, manuales o diseños que rigen una implementación.
- **ENT SYS DOCUMENTO TRIBUTARIO:** Archivo con validez ante el SII (facturas, boletas) cargado en el sistema.
- **ENT SYS EMERGENCIA:** Estado del sistema que activa protocolos de contingencia y alta disponibilidad.
- **ENT SYS ENTIDAD DATOS:** Objeto semántico que representa un concepto del negocio dentro del sistema.
- **ENT SYS ENTORNO DESPLIEGUE:** Infraestructura (Dev, Test, Prod) donde reside el código del sistema.
- **ENT SYS ENTORNO INFRAESTRUCTURA:** Conjunto de servidores, redes y storage que soportan GORE_OS.
- **ENT SYS EVENTO:** Cualquier suceso relevante que gatilla una acción o registro en el sistema.
- **ENT SYS EVIDENCIA DIGITAL:** Dato o archivo que puede ser usado para acreditar la realización de un acto.
- **ENT SYS EXPEDIENTE CONFIDENCIAL:** Caso con acceso restringido por contener datos sensibles o ley especial.
- **ENT SYS EXPEDIENTE ELECTRONICO:** Carpeta digital trazable que consolida un proceso administrativo completo.
- **ENT SYS EXPEDIENTE JUDICIAL:** Registro del flujo de documentos y actuaciones en causas legales del GORE.
- **ENT SYS EXPEDIENTE TECNICO:** Consolidación de diseños y aprobaciones de una iniciativa de inversión.
- **ENT SYS FEATURE FLAG:** Interruptor lógico que permite activar o desactivar funciones del sistema en tiempo real.
- **ENT SYS FEED VIDEO:** Flujo continuo de datos audiovisuales proveniente de una cámara de seguridad.
- **ENT SYS FILE:** Unidad física de almacenamiento de datos en el sistema de archivos.
- **ENT SYS FORMULARIO:** Interfaz de captura estructurada de datos para procesos administrativos.
- **ENT SYS FORMULARIO MOVIL:** Interfaz de captura optimizada para dispositivos en terreno (tablets/celulares).
- **ENT SYS GOBERNANZA DATOS:** Políticas y roles para asegurar que el dato sea un activo confiable.
- **ENT SYS GOLDEN RECORD:** Versión única y verídica de un dato tras resolver conflictos de múltiples fuentes.
- **ENT SYS GRABACION EVIDENCIA:** Archivo de audio o video resultante de la captura de un suceso relevante.
- **ENT SYS GRUPO PILOTO:** Conjunto de usuarios que prueban funciones antes del despliegue masivo.
- **ENT SYS IAC TEMPLATE:** Plantilla de Infraestructura como Código para orquestar recursos en la nube.
- **ENT SYS INFORME TECNICO:** Documento de análisis generado por el sistema a partir de datos procesados.
- **ENT SYS INDICADOR TECNICO:** Variable de monitoreo del rendimiento de la plataforma informática.
- **ENT SYS INFRAESTRUCTURA TI:** Recursos de hardware y conectividad que sustentan la operación del GORE.
- **ENT SYS INSCRIPCION:** Acto de registro de un actor o entidad en un programa o servicio digital.
- **ENT SYS INTEROPERABILIDAD NACIONAL:** Capacidad de intercambio de datos con plataformas centrales del Estado.
- **ENT SYS INTEROPERABILIDAD SIAPER:** Conexión automática con el sistema de personal de la Contraloría.
- **ENT SYS LISTA PRIORIZADA:** Ordenación de elementos basada en algoritmos de puntuación o decisión.
- **ENT SYS LLAVE PRIVADA:** Componente del par criptográfico usado para firmar digitalmente.
- **ENT SYS LMS PLATAFORMA:** Sistema de gestión del aprendizaje para la capacitación de funcionarios.
- **ENT SYS LOG:** Registro automático de sucesos en el servidor o aplicación.
- **ENT SYS LOG AUDITORIA:** Trazabilidad completa de quién, cuándo y qué hizo con un dato sensible.
- **ENT SYS LOG ERROR:** Registro de fallas técnicas para diagnóstico y reparación.
- **ENT SYS LOG TRANSACCION:** Asiento de movimientos que alteran el estado financiero o administrativo.
- **ENT SYS LOGS CAMBIOS:** Historial de modificaciones sobre la configuración del sistema.
- **ENT SYS LOGS ERRORES:** Acumulación de avisos técnicos para el monitoreo de salud de la plataforma.
- **ENT SYS LOGS QUERIES:** Registro de las consultas realizadas a la base de datos para análisis de performance.
- **ENT SYS LOGS SISTEMA:** Traza general de la actividad de los servicios de backend.
- **ENT SYS METADATO EXPEDIENTE:** Información adicional que describe y clasifica un expediente electrónico.
- **ENT SYS METRICA CICLO:** Tiempo promedio de tramitación de un proceso de punta a punta.
- **ENT SYS METRICA PERFORMANCE:** Dato sobre el rendimiento técnico de la plataforma (latencia, carga).
- **ENT SYS MODELO DATOS:** Representación lógica y relacional del universo de información del GORE.
- **ENT SYS NOTIFICACION:** Mensaje directo al usuario sobre un suceso que requiere su atención.
- **ENT SYS NOTIFICATION:** Aviso técnico generado por el sistema hacia otros servicios.
- **ENT SYS OBJETO APRENDIZAJE:** Unidad mínima de contenido educativo digital cargada en el LMS.
- **ENT SYS ONTOLOGIA:** Marco semántico que define las relaciones y conceptos del dominio regional.
- **ENT SYS ORDEN FIFO:** Lógica de atención donde el primero en ingresar es el primero en ser procesado.
- **ENT SYS PEDIDO MANTENCION:** Ticket interno para la reparación de un activo de TI o infraestructura.
- **ENT SYS PERMISO:** Regla que habilita o restringe una acción de un usuario en un módulo.
- **ENT SYS PIPELINE:** Secuencia automatizada de pasos para procesar datos o desplegar código.
- **ENT SYS PIPELINE VALIDACION:** Flujo automático que verifica la integridad y calidad de los datos cargados.
- **ENT SYS PLAN CONTINGENCIA:** Protocolo técnico para restaurar la operación ante una falla crítica.
- **ENT SYS PLANTILLA:** Modelo predefinido de documento o interfaz para estandarizar la captura.
- **ENT SYS PLANTILLA REPORTE:** Estructura base para la generación de informes recurrentes.
- **ENT SYS PLATAFORMA CIES:** Sistema regional de televigilancia y gestión de incidentes de seguridad.
- **ENT SYS PLATAFORMA OBSERVABILIDAD:** Herramienta que permite monitorear el estado interno del sistema GORE_OS.
- **ENT SYS PROBLEMA ESCALADO:** Suceso técnico que requiere intervención de niveles superiores de soporte.
- **ENT SYS PROCESO DIGITALIZADO:** Flujo administrativo antes manual que ahora corre íntegramente en el sistema.
- **ENT SYS PROTOCOLO EMERGENCIA:** Pasos técnicos a seguir ante desastres que afecten la infraestructura de TI.
- **ENT SYS PROTOCOLO ESCALAMIENTO:** Reglas para derivar casos o alertas a roles de mayor jerarquía.
- **ENT SYS PROTOCOLO RECUPERACION:** Procedimiento para restaurar datos desde respaldos tras un incidente.
- **ENT SYS PROTOCOLO RESPUESTA:** Acción estandarizada del sistema ante un tipo específico de alerta.
- **ENT SYS REGISTRO DPO:** Asiento de las actividades del Oficial de Protección de Datos.
- **ENT SYS REPORTE INTEGRIDAD:** Certificación automática de que los datos no han sido vulnerados.
- **ENT SYS REPOSITORIO:** Lugar de almacenamiento centralizado de código o documentos.
- **ENT SYS REPOSITORIO DAM:** Digital Asset Management; gestión centralizada de activos multimedia.
- **ENT SYS REPOSITORIO MULTIMEDIA:** Almacenamiento especializado para grandes volúmenes de video y audio.
- **ENT SYS RESERVA:** Asignación temporal de un recurso para un fin específico.
- **ENT SYS RESERVA RECURSO:** Bloqueo de disponibilidad de salas, móviles o equipos en el sistema.
- **ENT SYS ROL SISTEMA:** Conjunto de privilegios asignados a un perfil de usuario.
- **ENT SYS SCHEMA:** Estructura formal que define la organización de las tablas en una base de datos.
- **ENT SYS SIGBIE:** Sistema de Información de Gestión de Bienestar; módulo especializado.
- **ENT SYS SISTEMA MUNICIPAL:** Plataforma externa de las municipalidades con la que el GORE interopera.
- **ENT SYS SOLICITUD BENEFICIO:** Petición digital de un funcionario al programa de bienestar regional.
- **ENT SYS SOLICITUD LOGISTICA:** Petición interna de suministros o servicios operativos al área backoffice.
- **ENT SYS STORAGE ENV:** Definición de repositorios de almacenamiento para distintos entornos operativos.
- **ENT SYS TAG DOCUMENTAL:** Etiqueta semántica para facilitar la clasificación y búsqueda de archivos.
- **ENT SYS TEST AUTOMATICO:** Guion de pruebas que valida el funcionamiento del código sin intervención humana.
- **ENT SYS TICKET SOPORTE:** Registro de un requerimiento técnico o falla reportado por un usuario.
- **ENT SYS TRANSFERENCIA RESPONSABILIDAD:** Registro oficial del paso de un activo o caso de un actor a otro.
- **ENT SYS TRAZABILIDAD:** Capacidad de reconstruir la historia de un dato o proceso administrativo.
- **ENT SYS TRAZABILIDAD DECISION:** Registro de los fundamentos y responsables de una resolución administrativa.
- **ENT SYS USUARIO:** Cuenta individual que permite el acceso identificado a la plataforma GORE_OS.
- **ENT SYS VOTO ELECTRONICO:** Sistema seguro para consultas ciudadanas o decisiones de cuerpos colegiados.
