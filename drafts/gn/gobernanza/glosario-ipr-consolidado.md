# Glosario Consolidado IPR - GORE Ñuble
## Terminos
-
  ### Nombre
  1er Llamado
  ### Def
  Primer ciclo formal del año calendario para la presentación de proyectos a fondos concursables (como el 8% FNDR).
-
  ### Nombre
  2do Llamado
  ### Def
  Segundo ciclo de postulación a fondos concursables, activado según disponibilidad presupuestaria remanente.
-
  ### Nombre
  8% FNDR
  ### Def
  Subvención del Fondo Nacional de Desarrollo Regional destinada a financiar actividades de la comunidad en áreas como Cultura, Deporte, Seguridad y Social.
-
  ### Nombre
  A - Desarrollo Territorial
  ### Def
  Eje estratégico de la Estrategia Regional de Desarrollo (ERD) enfocado en la infraestructura y ordenamiento del territorio.
-
  ### Nombre
  ACB
  ### Def
  Análisis Costo-Beneficio: Metodología de evaluación social aplicada cuando los beneficios del proyecto son medibles y valorables en dinero.
-
  ### Nombre
  ACE
  ### Def
  Análisis Costo-Eficiencia: Metodología de evaluación que compara alternativas según su costo por unidad de beneficio no monetario (ej. costo por alumno).
-
  ### Nombre
  AD
  ### Def
  Admisibilidad - Estado que acredita cumplimiento mínimo para control y registro sin evaluación socioeconómica completa.
-
  ### Nombre
  ANF
  ### Def
  Activos No Financieros. Bienes durables que no son financieros (ej. edificios, maquinaria, vehículos).
-
  ### Nombre
  ANID
  ### Def
  Agencia Nacional de Investigación y Desarrollo: Institución colaboradora en la ejecución de fondos de innovación (FRPD/FIC).
-
  ### Nombre
  APU
  ### Def
  Análisis de Precios Unitarios: Desglose detallado del costo de materiales, mano de obra y equipos por cada partida de una obra.
-
  ### Nombre
  ATE
  ### Def
  Análisis Técnico-Económico: Proceso de evaluación ex-ante que determina la rentabilidad y viabilidad de una iniciativa de inversión.
-
  ### Nombre
  Acceso a financiamiento
  ### Def
  Capacidad de una institución para optar a recursos del presupuesto de inversión del Gobierno Regional.
-
  ### Nombre
  Acceso al Agua
  ### Def
  Línea de inversión prioritaria enfocada en soluciones hídricas, como sistemas de Agua Potable Rural (APR).
-
  ### Nombre
  Actividades
  ### Def
  Acciones específicas y cronológicas necesarias para producir los componentes de un proyecto o programa.
-
  ### Nombre
  Administración Directa
  ### Def
  Modalidad de ejecución donde el propio organismo (ej. Municipio) realiza las obras con su personal y equipos, sin contratar a un externo.
-
  ### Nombre
  Admisibilidad
  ### Def
  Revisión formal de completitud de antecedentes y cumplimiento de requisitos mínimos del instructivo y la Guía Operativa.
-
  ### Nombre
  Admisibilidad Documental
  ### Def
  Proceso específico de verificación de la existencia y validez formal de los documentos requeridos en la postulación.
-
  ### Nombre
  Admisibilidad FRIL
  ### Def
  Etapa específica del Fondo FRIL donde se chequea que la iniciativa cumpla con los montos, tipologías y requisitos del instructivo anual.
-
  ### Nombre
  Admisible para financiamiento (Conservación)
  ### Def
  Resultado favorable de evaluación MDSF para proyectos de conservación; habilita financiamiento sin ser RS.
-
  ### Nombre
  Ahorro Tributario
  ### Def
  Beneficio contable considerado en los flujos de caja privados, derivado de la depreciación o gastos financieros.
-
  ### Nombre
  Analista GORE
  ### Def
  Profesional de la División de Planificación o Presupuesto encargado de la revisión técnica y financiera de las iniciativas.
-
  ### Nombre
  Anexos: formularios y documentos tipo
  ### Def
  Conjunto de plantillas estandarizadas por el GORE para la formulación y postulación de iniciativas.
-
  ### Nombre
  Análisis Independiente
  ### Def
  Revisión técnica de una iniciativa realizada por un evaluador distinto al formulador para asegurar imparcialidad.
-
  ### Nombre
  Análisis Separado
  ### Def
  Evaluación técnica de componentes específicos de un proyecto que requieren un tratamiento metodológico diferenciado.
-
  ### Nombre
  Análisis Técnico-Económico (ATE)
  ### Def
  Evaluación que combina la factibilidad técnica con la rentabilidad social o costo-eficiencia de un proyecto.
-
  ### Nombre
  Análisis Unificado
  ### Def
  Evaluación integral de una iniciativa que contempla todos sus componentes como un bloque indivisible.
-
  ### Nombre
  Análisis causal y de efectos
  ### Def
  Metodología diagnóstica utilizada en la formulación de proyectos para identificar el problema central y sus impactos.
-
  ### Nombre
  Aporte Propio
  ### Def
  Recursos financieros o valorizados que la entidad postulante compromete para complementar el financiamiento del GORE.
-
  ### Nombre
  Articulación de actores
  ### Def
  Coordinación entre instituciones públicas y privadas para potenciar el impacto de una Intervención Pública Regional.
-
  ### Nombre
  Atingencia del programa
  ### Def
  Grado de coherencia entre los objetivos de un programa y el problema público que pretende resolver.
-
  ### Nombre
  Atribuciones del Gobernador Regional
  ### Def
  Facultades legales otorgadas a la máxima autoridad regional para administrar recursos y dirigir la gestión del GORE.
-
  ### Nombre
  B - Servicios
  ### Def
  Categoría de inversión o eje enfocado en la provisión y mejora de prestaciones básicas a la ciudadanía.
-
  ### Nombre
  BIP
  ### Def
  Banco Integrado de Proyectos. Plataforma informática del SNI donde se registran y gestionan las iniciativas de inversión pública.
-
  ### Nombre
  BIPS
  ### Def
  Banco Integrado de Programas Sociales: Registro del Ministerio de Desarrollo Social para la oferta programática social.
-
  ### Nombre
  BNUP
  ### Def
  Bien Nacional de Uso Público: Inmuebles cuyo dominio pertenece a la nación toda (calles, plazas) y son administrados por municipios.
-
  ### Nombre
  BPC
  ### Def
  Base de Personas Cuidadoras: Registro administrativo utilizado para focalizar programas de apoyo a cuidadores no remunerados.
-
  ### Nombre
  Bifurcación de Evaluación
  ### Def
  Punto en el flujo del FRPD donde se decide si un proyecto se evalúa por criterios de Ciencia (CTCI) o Fomento Productivo tradicional.
-
  ### Nombre
  Biodiversidad y Senderos
  ### Def
  Línea específica del fondo 8% Medio Ambiente orientada a la conservación de ecosistemas y habilitación de rutas naturales.
-
  ### Nombre
  C - Desarrollo Social y Económico
  ### Def
  Eje estratégico de la ERD centrado en el bienestar humano y el fomento productivo regional.
-
  ### Nombre
  CAE
  ### Def
  Costo Anual Equivalente. Indicador utilizado en evaluación de proyectos para comparar alternativas con vidas útiles diferentes.
-
  ### Nombre
  CDP
  ### Def
  Certificado de Disponibilidad Presupuestaria: Documento que garantiza que existen fondos libres para comprometer un gasto.
-
  ### Nombre
  CDR
  ### Def
  Comité Directivo Regional: Instancia estratégica del GORE que valida la pertinencia de programas e iniciativas mayores.
-
  ### Nombre
  CGR
  ### Def
  Contraloría General de la República: Órgano fiscalizador que controla la legalidad de los actos y el gasto público.
-
  ### Nombre
  CIES
  ### Def
  Centro Integral de Emergencia y Seguridad: Unidad técnica regional coordinadora en materias de prevención y desastres.
-
  ### Nombre
  CLP
  ### Def
  Peso Chileno. Moneda oficial de Chile.
-
  ### Nombre
  CORE
  ### Def
  Consejo Regional: Cuerpo colegiado que aprueba el presupuesto, los instructivos y la distribución de los fondos regionales.
-
  ### Nombre
  CORFO
  ### Def
  Corporación de Fomento de la Producción: Agencia ejecutora clave en programas de emprendimiento e innovación.
-
  ### Nombre
  COSOC
  ### Def
  Consejo de la Sociedad Civil: Mecanismo de participación ciudadana que puede observar la gestión del GORE.
-
  ### Nombre
  CTCI
  ### Def
  Ciencia, Tecnología, Conocimiento e Innovación: Ámbitos estratégicos financiados por el fondo FIC/FRPD.
-
  ### Nombre
  Calidad Deficiente
  ### Def
  Estado de una formulación que no cumple con los estándares técnicos mínimos requeridos para su evaluación.
-
  ### Nombre
  Calidad y pertinencia
  ### Def
  Criterios de evaluación que miden la excelencia técnica y la adecuación de una propuesta al contexto local.
-
  ### Nombre
  Carpeta Digital
  ### Def
  Repositorio electrónico en BIP para toda la documentación de la IDI.
-
  ### Nombre
  Carta Gantt Financiera
  ### Def
  Cronograma que planifica la ejecución del gasto (flujo de caja) mes a mes durante la vida del proyecto.
-
  ### Nombre
  Carta de Aceptación de Términos y Condiciones
  ### Def
  Documento legal donde el postulante declara conocer y aceptar las reglas del concurso o financiamiento.
-
  ### Nombre
  Carta de Compromiso de Prevención (violencia, drogas y alcohol)
  ### Def
  Declaración obligatoria en subvenciones para garantizar ambientes seguros durante la ejecución de actividades.
-
  ### Nombre
  Carta de Patrocinio
  ### Def
  Documento oficial donde una autoridad institucional respalda una postulación, comprometiendo apoyo institucional.
-
  ### Nombre
  Carta de Presentación y Experiencia de la Organización
  ### Def
  Documento que acredita la trayectoria y capacidad de la entidad postulante para ejecutar el proyecto.
-
  ### Nombre
  Carta de Solicitud de Recursos
  ### Def
  Oficio formal dirigido al Gobernador Regional mediante el cual se ingresa oficialmente una petición de financiamiento.
-
  ### Nombre
  Caución
  ### Def
  Garantía financiera (boleta, póliza) exigida para asegurar el correcto uso de los recursos anticipados o transferidos.
-
  ### Nombre
  Centro Integral de Emergencia y Seguridad
  ### Def
  Unidad del GORE Ñuble responsable de la gestión del Fondo de Seguridad Ciudadana del Concurso 8% y de la coordinación técnica asociada.
-
  ### Nombre
  Certificado de Acreditación CNA
  ### Def
  Documento que certifica la calidad institucional de una universidad, requisito para postular a ciertos fondos FRPD.
-
  ### Nombre
  Certificado de Baja
  ### Def
  Documento que acredita que un activo (vehículo, equipo) ha sido retirado del servicio, requisito para financiar su reposición.
-
  ### Nombre
  Certificado de Compromiso Financiero
  ### Def
  Documento donde la entidad (ej. Municipio) se obliga a financiar los costos de operación y mantención futuros del proyecto.
-
  ### Nombre
  Certificado de Directorio Vigente
  ### Def
  Acreditación de la personalidad jurídica y representantes actuales de una organización privada.
-
  ### Nombre
  Certificado de Dominio Vigente
  ### Def
  Documento del Conservador de Bienes Raíces que prueba la propiedad del terreno donde se emplazará el proyecto.
-
  ### Nombre
  Certificado de Factibilidad
  ### Def
  Documento de empresas de servicios (luz, agua) que asegura la posibilidad técnica de conexión para el proyecto.
-
  ### Nombre
  Certificado de No Fraccionamiento
  ### Def
  Declaración jurada de que el proyecto no ha sido dividido para evadir el sistema de evaluación de inversiones.
-
  ### Nombre
  Certificado de No Fraccionamiento de Programas
  ### Def
  Declaración que asegura que una iniciativa no ha sido dividida artificialmente para eludir controles o límites montos.
-
  ### Nombre
  Certificado de Pertinencia
  ### Def
  Validación estratégica otorgada por el GORE indicando que una iniciativa se alinea con la Estrategia Regional de Desarrollo.
-
  ### Nombre
  Certificado de Pertinencia y Patrocinio GORE
  ### Def
  Documento que avala que una iniciativa de un tercero es de interés estratégico para el Gobierno Regional.
-
  ### Nombre
  Certificados de ensayos de materiales
  ### Def
  Documentación técnica que acredita la calidad de los componentes utilizados en obras de infraestructura.
-
  ### Nombre
  Ciencia, Tecnología, Conocimiento e Innovación
  ### Def
  Conjunto de actividades de generación y aplicación de conocimiento que el FRPD prioriza como motor de transformación y desarrollo regional.
-
  ### Nombre
  Circular 33
  ### Def
  Instrucción del Ministerio de Hacienda que permite procesos simplificados para adquisición de activos y conservación.
-
  ### Nombre
  Circular 33 (ANF, Estudios)
  ### Def
  Mecanismo simplificado para financiar adquisición de activos y estudios del giro institucional sin evaluación MIDESO.
-
  ### Nombre
  Circular N° 11/2025 Ministerio de Hacienda
  ### Def
  Normativa nacional que imparte instrucciones sobre la ejecución presupuestaria del sector público para dicho año.
-
  ### Nombre
  Clubes de Adulto Mayor
  ### Def
  Organizaciones comunitarias funcionales de personas mayores, línea exclusiva de financiamiento en el 8% FNDR.
-
  ### Nombre
  Cobertura Insuficiente
  ### Def
  Deficiencia en el diseño de un proyecto donde la población beneficiaria no alcanza los niveles mínimos de impacto.
-
  ### Nombre
  Cofinanciamiento
  ### Def
  Aporte monetario obligatorio o voluntario de la entidad ejecutora que complementa el subsidio del GORE.
-
  ### Nombre
  Coherencia con políticas
  ### Def
  Grado de alineación entre una Intervención Pública Regional y los planes nacionales o regionales vigentes.
-
  ### Nombre
  Coherencia interna del diseño
  ### Def
  Relación lógica entre el problema, los objetivos, las actividades y los productos de una iniciativa.
-
  ### Nombre
  Comité Regional de Ciencia
  ### Def
  Instancia asesora que valida la alineación de proyectos CTCI con la estrategia de innovación regional.
-
  ### Nombre
  Comité de Pertinencia
  ### Def
  Instancia técnica del GORE que revisa si las solicitudes ingresadas cumplen con los lineamientos estratégicos antes de evaluarlas.
-
  ### Nombre
  Comité de Pertinencia Regional
  ### Def
  Órgano colegiado interno que prioriza las iniciativas de inversión según las definiciones del Gobernador y el CORE.
-
  ### Nombre
  Comité de Seguimiento
  ### Def
  Grupo de trabajo encargado de monitorear el avance físico y financiero de programas complejos (PPR).
-
  ### Nombre
  Componentes
  ### Def
  Productos, bienes o servicios específicos que entrega un proyecto y que son necesarios para lograr su propósito.
-
  ### Nombre
  Comunicación oficial de resultados
  ### Def
  Notificación formal del GORE sobre la aprobación, rechazo u observaciones de una postulación.
-
  ### Nombre
  Concurso 8%
  ### Def
  Fondo concursable de subvenciones del FNDR destinado a organizaciones sociales para cultura, deporte, seguridad y social.
-
  ### Nombre
  Concurso Vinculación con la Comunidad 8%
  ### Def
  Concurso anual de subvenciones FNDR hasta por el 8% de los recursos de inversión regional, destinado a financiar iniciativas de vinculación con la comunidad en diversos fondos temáticos.
-
  ### Nombre
  Conservación
  ### Def
  Intervención destinada a recuperar el estándar original de una infraestructura sin aumentar su capacidad o superficie.
-
  ### Nombre
  Consistencia del diseño
  ### Def
  Validación de que los elementos técnicos de un proyecto sean armónicos y factibles entre sí.
-
  ### Nombre
  Constitución Política de la República
  ### Def
  Norma suprema que define la estructura del Estado y las competencias generales de los niveles regionales.
-
  ### Nombre
  Contenido Mínimo del Convenio
  ### Def
  Cláusulas obligatorias que deben estar presentes en el acuerdo legal entre el GORE y la entidad ejecutora.
-
  ### Nombre
  Control jurídico adicional del GORE
  ### Def
  Revisión legal exhaustiva realizada por la Unidad Jurídica para asegurar la legalidad de actos administrativos complejos.
-
  ### Nombre
  Convenio de Transferencia
  ### Def
  Contrato administrativo entre el GORE y la entidad ejecutora que regula el traspaso y rendición de los recursos.
-
  ### Nombre
  Coordinación
  ### Def
  Actividad de gestión orientada a armonizar acciones de distintos actores para un fin común.
-
  ### Nombre
  Costo Marginal
  ### Def
  Costo adicional de producir una unidad más; en evaluación, se usa para decidir el momento óptimo de reemplazo de equipos.
-
  ### Nombre
  Cotizaciones de Respaldo
  ### Def
  Presupuestos de proveedores que justifican los montos solicitados en el diseño de una iniciativa.
-
  ### Nombre
  Código BIP
  ### Def
  Identificador numérico único asignado por el MIDESO a cada iniciativa de inversión pública registrada.
-
  ### Nombre
  D - Medio Ambiente
  ### Def
  Eje estratégico de la ERD enfocado en la sustentabilidad, protección de recursos naturales y cambio climático.
-
  ### Nombre
  DAE
  ### Def

-
  ### Nombre
  DAF

  ### Nombre

  ### Nombre

  ### Nombre

-
  ### Nombre
  DIPRES

-
  ### Nombre

-
  ### Nombre
  DOM

  ### Nombre

-
  ### Nombre

  Documento donde el postulante asegura estar al día con sus rendiciones en el sistema nacional.
-
  ### Nombre
  Declaración de Horas Disponibles
  ### Def
  Documento exigido a académicos/investigadores para asegurar que tienen tiempo real para dedicar al proyecto.
-
  ### Nombre

  ### Def
  Norma histórica que sentó las bases de la división política y administrativa de Chile en regiones.
-
  ### Nombre
  Decreto Ley N°1.263 (Adm. Financiera Estado)
  ### Def
  Ley orgánica que regula los procesos de presupuesto, contabilidad y tesorería del sector público.
-
  ### Nombre
  Departamento de Análisis y Evaluación
  ### Def
  Unidad del GORE responsable de la evaluación técnica de programas y proyectos.
-
  ### Nombre

  ### Nombre

  Dimensiones de desempeño evaluadas en los indicadores de los programas públicos.
-
  ### Nombre

  Línea de proyectos enfocada en optimizar el uso del agua y reducir pérdidas en sistemas de riego o consumo.

  ### Nombre

  ### Def

-
  ### Nombre
  Transferencia a Entidad Pública
  ### Def
  Traspaso de recursos desde el GORE a otro servicio (MOP, SERVIU, Municipio) para que ejecute una política regional.
-

  ### Nombre

  ### Nombre

-
  ### Nombre
