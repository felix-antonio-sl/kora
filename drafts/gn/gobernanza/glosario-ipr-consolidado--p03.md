## Términos R
| término | definición | aliases |
| --- | --- | --- |
| Ranking de Elegibilidad | Lista ordenada de proyectos según su puntaje de evaluación técnica, usada para asignar recursos limitados. | |
| RATE | Resultado del Análisis Técnico-Económico: Dictamen formal (RS, OT, FI) emitido tras la evaluación del proyecto. | |
| Recepción Provisoria / Definitiva | Actos administrativos donde se certifica que la obra ha sido terminada conforme a contrato (con y sin garantías vigentes). | |
| Recomendado Favorablemente | Calificación que indica aprobación técnica y habilita la obtención de RS para financiamiento. | |
| Reconstrucción | Recuperación de infraestructura dañada por catástrofes; puede postularse vía Circular 33 como inversión. | |
| Reconstrucción / Emergencia | Descriptor o etiqueta en el BIP para proyectos priorizados que responden a desastres naturales. | |
| Redistribución de Excedentes | Facultad del GORE para reasignar recursos remanentes de una línea del 8% a otra con mayor demanda. | |
| Reducción de brechas y equidad | Objetivo central de la inversión regional para disminuir desigualdades entre comunas y sectores. | |
| Reevaluación | Nuevo análisis de un proyecto ya aprobado (RS) que ha sufrido cambios sustanciales de costo (mayor al 10%) o plazo. | |
| Registro de Colaboradores del Estado | Padrón nacional obligatorio para todas las entidades privadas que reciben transferencias públicas. | |
| Registro Social de Hogares (RSH) | Herramienta fundamental para la focalización de programas de ayuda social regional. | |
| Regla del 30% | Norma de la Circular 33: si la conservación cuesta menos del 30% de la reposición, se evalúa simplificadamente. | |
| Reglas sobre subcontratación | Límites y obligaciones que debe cumplir la entidad ejecutora cuando delega parte del proyecto a terceros. | |
| Relación entre planos y EE.TT | Consistencia obligatoria entre la representación gráfica y la descripción técnica de una obra. | |
| Remuneraciones | Sueldos imponibles del personal contratado para el proyecto, sujetos a topes máximos según normativa. | |
| Representación | Financiamiento para delegaciones o individuos que compiten o exponen a nombre de la región en eventos externos. | |
| Representante Legal | Persona facultada estatutariamente para actuar y firmar convenios a nombre de la institución postulante. | |
| Requisitos de Información Sectorial (RIS) | Manuales del SNI que detallan los documentos adicionales según el tipo de proyecto (educación, salud, etc.). | |
| Residencias Mejor Niñez | Residencias de protección del Servicio Nacional de Protección Especializada a la Niñez y Adolescencia, con línea de financiamiento específica dentro del Fondo Social. | |
| Resolución 30 CGR | Normativa de la Contraloría General que fija las normas sobre rendición de cuentas de fondos públicos. | |
| Resolución Exenta | Acto administrativo del GORE que no requiere Toma de Razón inmediata (generalmente), usado para aprobar convenios y bases. | |
| Resolución Exenta N° 15.051/2023 SUBDERE | Normativa nacional que regula la asignación de cupos y montos para el fondo FRIL. | |
| Resolución N° 36/2024 CGR | Instrucción de la Contraloría sobre la toma de razón de actos administrativos. | |
| Responsabilidad | Obligación de la entidad receptora de fondos de cumplir con los objetivos y rendir el gasto correctamente. | |
| Respuesta pertinente al territorio | Grado en que la solución propuesta se adapta a la realidad geográfica y cultural de la comuna beneficiaria. | |
| Restitución de Fondos | Obligación de devolver los recursos transferidos no ejecutados o no rendidos satisfactoriamente. | |
| Resultado de la Evaluación Técnica | Clasificaciones RS, FI, OT y NV que resumen el resultado de la evaluación técnica de una iniciativa. | |
| RF | Recomendación Favorable: Calificación de aprobación técnica aplicada a programas sociales (equivalente a RS). | |
| RF – Recomendado Favorablemente | Calificación positiva otorgada a iniciativas que demuestran beneficios sociales superiores a sus costos. | |
| RIS | Requisitos de Información Sectorial: Set de antecedentes técnicos específicos exigidos según el tipo de proyecto (Salud, Vialidad). | |
| Royalty Minero | Impuesto a la minería que genera un fondo de compensación distribuido a las regiones para innovación y desarrollo. | |
| RS | Recomendado Satisfactoriamente - Aprobado financiamiento | |
| RSH | Registro Social de Hogares: Sistema de información que apoya la selección de beneficiarios según su nivel socioeconómico. | |

## Términos T
| término | definición | aliases |
| --- | --- | --- |
| Tasa Social de Descuento | Porcentaje utilizado para traer a valor presente los flujos futuros (actualmente 6%), reflejando el costo de oportunidad social. | |
| TDR | Términos de Referencia: Documento que define los objetivos, alcances y productos esperados de un estudio o consultoría. | |
| Tenencia Responsable | Línea de financiamiento (8% o programas) para control de población animal, vacunación y educación sanitaria. | |
| TIR | Tasa de descuento que hace que el VAN social sea igual a cero (rentabilidad social intrínseca del proyecto). | |
| Toma de Razón | Control preventivo de legalidad realizado por la Contraloría sobre decretos y resoluciones de alto monto. | |
| Transferencia a Entidad Pública | Traspaso de recursos desde el GORE a otro servicio (MOP, SERVIU, Municipio) para que ejecute una política regional. | |
| Transferencia Consolidable | Movimiento de fondos entre entidades del Gobierno Central que se compensa en la contabilidad nacional. | |
| Transferencia Directa (No Consolidable) | Entrega de fondos a privados (OSC) que no se consolida con el resto del sector público. | |
| Transferencia Municipal | Aportes entregados a municipios (ej. FRIL) que contablemente son gastos para el GORE y no se consolidan. | |
| Transferencia No Consolidable | Transferencia a entidades fuera del Presupuesto del Gobierno Central (ej. Privados, Universidades, Municipios). | |
| Transferencia Tecnológica | Proceso de transmisión de conocimientos científicos y tecnologías desde centros de I+D hacia empresas o usuarios. | |
| Transparencia y publicidad | Deber de los organismos públicos de dar a conocer sus actos y fundamentos a cualquier ciudadano. | |
| Trato Directo | Mecanismo excepcional de contratación sin licitación, permitido solo bajo causales estrictas y fundadas de la ley. | |
| Turismo | Área de inversión orientada a potenciar los atractivos regionales como motor de desarrollo económico. | |
