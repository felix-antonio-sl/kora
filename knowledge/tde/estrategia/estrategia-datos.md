# Estrategia de Datos de la Administración del Estado

Documento de lineamientos y hoja de ruta para gestión estratégica de datos con foco en valor público. Emitido por Secretaría de Gobierno Digital (SGD). Plan de acción actualizable cada 6 meses.

## Contexto y justificación

### Posición Chile

- Líder regional TD: 2do en LAC tras Uruguay (E-Government Survey 2024, Naciones Unidas)
- ClaveUnica: >15 millones usuarios
- Digitalización compras públicas, solicitudes info pública, tramitación administrativa
- Plataformas clave: ClaveUnica, FirmaGob, SIMPLE, SII tributario, ChileAtiende, RIS (MDS)

### Brechas identificadas

**Estructurales:**
- No existen estructuras gobernanza datos consolidadas para todo el Estado
- SGD recién creada (Ley 21.658) con rol coordinador uso estratégico datos
- Silos y fragmentación datos en Administración Pública
- Solo 1% municipios habilitados en red interoperabilidad
- 23% instituciones con áreas dedicadas gestión datos
- 10,9% con estrategia clara en datos
- 31,1% instituciones participa activamente Portal Datos Abiertos
- 54,8% aplica procedimientos anonimización
- Correo electrónico = medio predominante intercambio info (65,7% municipalidades, 85,3% instituciones regionales)
- Solo 29,3% municipalidades usa servicios web para compartir datos

**Talento humano:**
- Escasez profesionales TIC sector público (especialmente perfiles datos)
- Municipios: sin planes formación específicos ni identificación talentos digitales
- Resistencia al cambio por falta alfabetización digital
- Brechas urbano-rurales en capacidades municipales

**Tecnológicas:**
- Sistemas obsoletos sin garantías seguridad
- Falta infraestructura, arquitectura info/datos, protocolos seguridad
- Bajo uso tecnologías avanzadas interoperabilidad
- Comunas rurales: menor equipamiento, equipos antiguos

### Marco normativo vinculante

- Ley 21.658 -- crea SGD (coordinación uso estratégico datos)
- Ley 21.663 -- Marco Ciberseguridad (estándares seguridad infraestructura crítica)
- Ley protección datos personales (nuevo marco + estándar GDPR)
- Ley 21.180 -- TD (interoperabilidad + principio "solo una vez")

## Visión 2030

Administración del Estado completamente impulsada por datos, información y conocimiento. Servicios proactivos, eficientes, seguros y confiables. Chile líder OCDE en uso estratégico de datos. Referente regional en gobernanza digital.

## Objetivo

Transformar la Administración del Estado mediante uso estratégico de datos, información y conocimiento para mejorar calidad de vida, responder eficiente y proactivamente a necesidades/desafíos, generar valor público.

## Ejes estratégicos

### Eje 1 -- Ecosistema: gestión y gobernanza de datos

**Lineamientos:**

1. **Gestión con enfoque ciudadano**: gestión datos a lo largo de ciclo de vida centrada en necesidades/derechos ciudadanos. Cada OAE = fuente confiable de datos en su competencia. Enfoque coordinado: datos entendidos, intercambiados y procesados por toda la Administración.

2. **Coordinación multinivel**: intercambio datos en todos los niveles (especial apoyo gobiernos locales). Identificar/optimizar flujos intercambio. Eliminación progresiva duplicación esfuerzos/información.

3. **Coordinación intersectorial**: todos los sectores capaces de interoperar e integrar datos. Priorización apoyo estratégico sectores clave. Interoperabilidad estandarizada vía soluciones comunes. Formulación políticas más globales.

**Medidas plan de acción:**
- Consejo Estratégico de Datos de la Administración del Estado
- Regulación designación encargados de datos en OAE
- Grupos técnicos directores/as de datos
- Fortalecimiento gestión/gobernanza datos en sectores estratégicos
- Incentivo específico gobernanza datos
- Instancias colaboración público-privadas
- Inventario de datos del Estado
- Normativas técnicas estandarización datos maestros/referencia
- Plataformas datos transversales + analítica avanzada
- Alianzas actores tecnológicos estratégicos

### Eje 2 -- Personas: habilidades y capacidades

**Lineamientos:**

1. **Desarrollo talento digital**: aprendizaje continuo, capacitación y especialización para todo el personal. Reducir brechas, retener talento digital. Competencias adecuadas, oportunas, correctas.

2. **Gestión del cambio**: marco competencias para acceso/comprensión infraestructura digital habilitante. Identificación/participación activa trabajadores en procesos cambio. Cumplimiento normativa ciberseguridad, privacidad, gobierno digital.

3. **Mejora gestión y servicios**: empoderamiento/alfabetización trabajadores sector público. Tecnología potencia capacidades humanas. Innovación como herramienta desarrollo/modernización empleo público.

**Medidas plan de acción:**
- Capacitaciones y certificaciones gobierno/gestión datos
- Apoyo reclutamiento perfiles datos en instituciones públicas
- Cursos: IA, gobierno datos, gestión datos, interoperabilidad, datos abiertos
- Capacitación plataforma integración datos
- Promoción adopción herramientas

### Eje 3 -- Tecnología: infraestructura pública digital

**Lineamientos:**

1. **Red de Interoperabilidad**: fortalecimiento/consolidación en todos niveles/sectores. Principio "solo una vez". Servicios centralizados/compartidos, estándares comunes. Mejora continua seguridad, resiliencia, eficiencia.

2. **Datos abiertos**: motor innovación/desarrollo económico. Uso, reutilización, redistribución libre. Fortalecimiento infraestructura acceso/encontrabilidad. APIs + herramientas código abierto.

3. **Integración de datos**: servicios/infraestructuras integradas. Acceso, cruce, vinculación datos actualizados, relevantes, confiables. Evaluación políticas públicas, investigación interés público, investigación científica.

4. **Innovación e IA centrada en datos**: infraestructura para implementación tecnologías emergentes. Gestión riesgos asociados (Política Nacional IA). Sandboxes para análisis resultados/riesgos/sesgos. Bases datos usables por IA. Transparencia datos entrenamiento modelos.

**Medidas plan de acción:**
- Portal público avance estrategia datos
- Página web gobierno de datos
- Mejoras portal datos abiertos
- Integración plataformas geoespaciales
- Nueva plataforma interoperabilidad Estado
- Mecanismos gestión consentimiento datos personales
- Transparencia algorítmica
- Guía pilotos IA
- Capacitaciones uso ético IA
- Centro de Excelencia IA (casos uso entornos controlados)

## Glosario clave

| Concepto | Definición |
|----------|-----------|
| Dato | Representación atributo/variable cuanti/cualitativa vía secuencia letras/números/símbolos |
| Datos abiertos | Dato digital con condiciones técnicas/jurídicas para uso, reutilización, redistribución libre |
| Datos maestros | Datos que proveen contexto a datos actividad negocio (definiciones, identificadores entidades) |
| Datos referencia | Conjuntos datos que categorizan/relacionan otros datos mediante códigos |
| Integración datos | Procesos movimiento/consolidación datos entre almacenes, aplicaciones, organizaciones |
| Interoperar | Comunicar/transferir datos entre instituciones en línea, mínima intervención humana, estándares comunes |
| Metadatos | Datos que describen contexto, contenido, estructura de un dato |
| Servicios compartidos | Servicios/herramientas prestados por AE usados por 2+ órganos |
| Servicios digitales | Prestaciones OAE a usuarios vía tecnologías digitales |
