# Glosario SIMPLE

Términos asociados a la plataforma SIMPLE (Sistema de Implementación de Procesos Ligeramente Estandarizados).

## A

| Término | Definición |
|---------|-----------|
| Acciones | Conjunto de funcionalidades ejecutables dentro de un proceso: enviar correo, generar variables, consultar servicios web, etc. |
| Acompañamiento equipo de digitalización | Servicio de la SGD mediante su equipo de digitalización para desarrollo de procesos institucionales. Actividades/requisitos en guía de servicio de acompañamiento. |
| Ambiente de desarrollo | Ambiente compartido para desarrollo de nuevos procesos de los OAE que usan SIMPLE. |
| Ambiente de capacitación | Ambiente compartido para manuales de capacitación y ejecución del taller Aprendiendo SIMPLE. |
| Ambiente productivo | Ambiente entregado a cada OAE por la SGD gratuitamente para disponibilizar procesos a las personas. |
| API | Application Programming Interface. Conjunto de reglas, protocolos y herramientas para comunicación entre aplicaciones de software. |
| Aprendiendo SIMPLE | Taller práctico del equipo de digitalización, parte de la fase de capacitación de funcionarios que trabajarán con SIMPLE. |
| Auditoría de procesos | Revisión y análisis de flujos de trabajo para garantizar cumplimiento de políticas y normativas. |
| Automatización de procesos | Uso de tecnología para realizar tareas y procesos con mínima intervención humana. |

## B

| Término | Definición |
|---------|-----------|
| Back-end | Parte del software que maneja lógica, procesamiento de datos e interacción con servidor. En SIMPLE: sección donde se encuentra el modelador de procesos. |
| BPMN | Business Process Model and Notation. Lenguaje estándar para modelar procesos de negocio con diagramas de flujo. |
| BPMS | Business Process Management System. Plataforma para modelar, automatizar, monitorear y optimizar procesos de negocio. |
| Buenas prácticas de digitalización | Recomendaciones entregadas a equipos institucionales mediante guía de Buenas prácticas de digitalización. |

## C

| Término | Definición |
|---------|-----------|
| Campo botón asíncrono | Ejecuta acciones en un mismo formulario sin presionar botón siguiente. |
| Campo botón siguiente | Reemplaza el botón siguiente por defecto. |
| Campo checkbox | Varias opciones de selección; permite marcar más de una. |
| Campo comunas | Despliega regiones y comunas del país para selección en formulario. |
| Campo date | Selección de fecha del calendario; puede restringirse según necesidades del OAE. |
| Campo documento | Referencia a documento creado en la plataforma. |
| Campo file | Subida de archivos. Máximo 15 MB. |
| Campo file transfer | Subida de archivos sin límite de tamaño. Vigencia: 30 días desde carga. |
| Campo grilla | Tabla editable con columnas configurables. |
| Campo grilla de datos externos | Campo para datos de fuentes externas. |
| Campo instituciones | Despliega nombres de OAE de Chile. |
| Campo Javascript | Permite agregar código Javascript al proceso. |
| Campo moneda | Listado de monedas de diferentes países. |
| Campo países | Listado de países del mundo. |
| Campo párrafo | Mayor número de caracteres; configurable con HTML. |
| Campo password | Input HTML tipo "password". |
| Campo provincias | Despliega regiones, provincias y comunas de Chile. |
| Campo radio | Varias opciones de selección; solo una marcable. |
| Campo select | Lista desplegable con varias opciones. |
| Campo subtítulo | Texto con etiqueta HTML "H4" por defecto; configurable (H1, H2, etc.). |
| Campo textarea | Área de texto multilínea para comentarios, descripciones, etc. |
| Campo textbox | Área de texto básica de una sola línea. |
| Campo título | Texto con etiqueta HTML "H3" por defecto; configurable (H1, H2, etc.). |
| Ciclo de vida de procesos | Fases en un BPMS: modelado → implementación → ejecución → monitoreo → optimización. |
| Consultar REST (acción) | Configuración para consumir servicio web REST + almacenar respuesta en variable. |
| Consultar SOAP (acción) | Configuración para consumir servicio web SOAP + almacenar respuesta en variable. |
| Crear conexión de unión | Conexión requerida con conexión paralela; permite pasar a siguiente tarea al completar todas las tareas paralelas. |
| Crear conexión paralela | Ejecuta más de una tarea simultáneamente en un proceso. |
| Crear conexión paralela por evaluación | Ejecuta tareas paralelas condicionalmente. |
| Crear conexión por evaluación | Avance condicional a una u otra tarea. |
| Crear conexión secuencial | Marca la siguiente tarea a ejecutar. |
| Crear tarea | Genera componente "tarea" con pasos, acciones y configuraciones para ejecución del proceso. |
| Cuenta | Instancia cloud de la institución en SIMPLE. Requiere administrador designado como contraparte del Equipo de Digitalización. |

## D

| Término | Definición |
|---------|-----------|
| Descargar documento (acción) | Descarga documento de URL pública y transforma en base64 dentro de variable en SIMPLE. |
| Diagrama de proceso | Representación visual de etapas, pasos y decisiones de un flujo de trabajo. |
| Digitalización de procesos | Transformación de procesos manuales en flujos automatizados mediante herramientas tecnológicas. |
| Diseñador | Módulo del backend de SIMPLE donde el digitalizador crea y configura tareas de un proceso. |
| Diseño/rediseño de procesos | Crear o modificar procesos para mayor eficiencia alineados a objetivos organizacionales. |
| Documentos | Archivos de texto generados mediante formato HTML dentro de SIMPLE. |

## E

| Término | Definición |
|---------|-----------|
| Enviar correo (acción) | Configuración de remitente, copia, copia oculta, asunto, contenido y adjuntos del correo. |
| Enviar documento (acción) | Envía documento generado en SIMPLE a endpoint institucional vía REST en formato base64. |
| Equipo de digitalización | Equipo de la SGD que brinda herramientas y asesoría para capacitar OAE en modelado y optimización de procesos. |

## F

| Término | Definición |
|---------|-----------|
| Formularios | Lienzo donde se agregan todos los campos de SIMPLE. |
| Front-end | Parte visible e interactiva de aplicación/sitio web. En SIMPLE: interfaz para funcionarios y personas usuarias. |

## G

| Término | Definición |
|---------|-----------|
| Generar documento (acción) | Genera documento dentro del trámite. Documento debe estar previamente elaborado en sección "documentos". |
| Generar variable (acción) | Genera nueva variable dentro del trámite en ejecución. |
| Gestión del cambio | Estrategia para adopción efectiva de procesos digitalizados en la organización. |
| Gestión de procesos | Disciplina de identificar, diseñar, documentar, implementar, monitorear y optimizar procesos de negocio. |
| Guía de uso de SIMPLE SaaS | Contiene elementos necesarios para usar SIMPLE: requisitos y compromisos. |

## I

| Término | Definición |
|---------|-----------|
| Integración | Conexión de sistemas, aplicaciones y servicios externos al BPMS para intercambio de información y ejecución de tareas. |

## K

| Término | Definición |
|---------|-----------|
| KPI | Key Performance Indicator. Métricas clave para evaluar desempeño y eficiencia de procesos. |

## M

| Término | Definición |
|---------|-----------|
| Manuales de digitalización | Manuales del Equipo de Digitalización para digitalizadores sobre SIMPLE. |
| Modelado de procesos | Representación gráfica de un proceso, comúnmente con BPMN. |
| Monitorización de procesos | Seguimiento en tiempo real del desempeño para identificar cuellos de botella o ineficiencias. |

## O

| Término | Definición |
|---------|-----------|
| Optimización de procesos | Mejora continua de flujos de trabajo para aumentar eficiencia y reducir costos. |

## P

| Término | Definición |
|---------|-----------|
| Presentación de procesos | Reunión post-acompañamiento donde digitalizadores del OAE presentan proceso al equipo de digitalización. |
| Procesos end-to-end | Procesos completos desde inicio hasta entrega del resultado final. |

## R

| Término | Definición |
|---------|-----------|
| Redirección (acción) | Redirige al usuario a URL externa a SIMPLE luego de completar tarea/paso. |
| Reglas de negocio | Directrices/condiciones que determinan acciones y decisiones dentro de un flujo. |

## S

| Término | Definición |
|---------|-----------|
| SaaS | Software as a Service. Aplicaciones alojadas en la nube ofrecidas vía internet. |
| SIMPLE Day | Webinar para conocer SIMPLE, sus usos, el acompañamiento y cómo acceder. |
| SIMPLE | Sistema de Implementación de Procesos Ligeramente Estandarizados. Sistema modelador de procesos (BPMS). |

## T

| Término | Definición |
|---------|-----------|
| Tareas de SIMPLE | Actividad configurada para modelado de procesos. |
| Task | Unidad mínima de trabajo dentro de un flujo en un BPMS. |

## V

| Término | Definición |
|---------|-----------|
| Variable | Dato/valor almacenado en la plataforma para un trámite específico. |

## W

| Término | Definición |
|---------|-----------|
| Workflow | Secuencia automatizada de tareas, reglas y decisiones que definen ejecución de un proceso. |
