# Guía de Evaluación de Calidad en Sitios Web y Servicios Digitales del Estado

Fecha: 27/12/2024, v.1.
Autor: Equipo de Servicios Digitales, SGD.
Base legal: Ley N° 21.658 (Secretaría de Gobierno Digital), Agenda de Modernización 2022-2026.

---

## 1. Objetivo

Marco de referencia práctico para evaluar y fortalecer calidad de sitios web y servicios digitales del Estado. Busca: mejorar experiencia de personas/empresas, garantizar estándares de calidad, apoyar OAE en desarrollo interno y contratación externa.

## 2. Definiciones

| Término | Definición |
|---------|-----------|
| Página web | Unidad de información específica del sitio |
| Plataforma electrónica | Software + datos + infraestructura que sustenta procesos |
| Servicio digital | Prestaciones OAE a usuarios mediante tecnologías digitales |
| Servicio digital transaccional | Requiere identificación para entrega de información personalizada o trámites |
| Sitio web | Conjunto de páginas bajo un mismo dominio |

## 3. Estructura del instrumento

### Secciones

| Sección | Contenido |
|---------|-----------|
| Portada | Presentación, método, datos evaluadores, escala, dimensiones, glosario |
| Muestra | URLs de páginas revisadas |
| Guía verificación técnica | Tutoriales para inspección de código, navegación teclado, herramientas |
| Dimensiones (hojas 1-20) | Indicadores y preguntas de chequeo por dimensión |
| Hoja de resultados | Puntaje ponderado por dimensión y nivel de obligatoriedad |
| Gráfico de resultados | Barras de % cumplimiento por dimensión y nivel |

### Niveles de prioridad

| Nivel | Descripción | Ponderación |
|-------|-------------|-------------|
| Imprescindible | Mínimos exigibles + obligatorios por marco legal. Ausencia = error/vulnerabilidad | 60% |
| Esperable | Condiciones esenciales de gobierno digital centrado en usuarios | 25% |
| Deseable | Valor agregado hacia excelencia | 15% |

### Escala de cumplimiento por indicador

| Puntaje | Descripción |
|---------|-------------|
| 0 | Ninguna respuesta afirmativa |
| 1 | <50% respuestas afirmativas |
| 2 | >=50% respuestas afirmativas |
| 3 | Todas las preguntas afirmativas |

## 6. Aplicación del instrumento

### Previo

- Familiarizarse con el instrumento.
- Solicitar colaboración de profesionales afines (comunicaciones, diseño, informática, transparencia, atención usuarios).

### Selección de muestra

**Sitios Web** (mínimo 10 páginas):
- Portada/página de inicio.
- Al menos 2 primeras páginas del menú principal (excluyendo portada).
- Al menos 3 páginas internas (una sobre trámites/servicios digitales si existe).
- 3 últimas noticias publicadas.
- Al menos 1 formulario de contacto sin autenticación.

**Servicios Digitales Transaccionales**:
- Página de acceso al trámite (landing page).
- Flujo completo del trámite en línea.

### Durante la evaluación

- Completar hoja "Inicio" (institución, URL, evaluador, fecha, navegador, dispositivo).
- Registrar muestra en hoja "Muestra".
- Contestar todas las preguntas (sin respuesta = 0 puntos).
- Umbral de cumplimiento: >= 50% de la muestra para respuesta afirmativa; si falla importante en una página = respuesta negativa.

## 7. Interpretación de resultados

**Cumplimiento total por dimensión**: promedio ponderado (Imprescindible 60%, Esperable 25%, Deseable 15%).

**Cumplimiento ponderado**: promedio de cada nivel considerando ponderación asignada a cada dimensión.

## 8. Recomendaciones de mejora

Priorización:
1. **Indicadores imprescindibles**: mínimos + obligatorios legales.
2. **Indicadores esperables**: mejoras importantes no críticas.
3. **Indicadores deseables**: valor agregado y optimización.

Dentro de cada nivel, priorizar dimensiones de mayor ponderación.
