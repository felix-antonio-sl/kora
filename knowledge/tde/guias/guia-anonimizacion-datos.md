# Guía Introductoria a la Anonimización de Datos

Fecha: 20/03/2025. Versión: v.1.
Autor: Secretaría de Gobierno Digital.
Formato: documento descargable (PDF).

---

## 1. Alcance

Introducción práctica a la anonimización de datos para organizaciones que gestionan bases de datos estructuradas. Cubre desde obtención hasta publicación de información, independiente del soporte o medio.

## 2. Contenido principal

### Conceptos fundamentales

- **Anonimización**: proceso irreversible que impide vincular datos con persona identificable.
- **Seudonimización**: sustitución de identificadores por seudónimos; reversible bajo condiciones controladas.
- **Reidentificación**: riesgo de vincular datos anonimizados con personas originales.

### Técnicas de anonimización

| Técnica | Descripción |
|---------|-------------|
| Enmascaramiento | Ocultar parcial o totalmente valores de datos sensibles |
| Aleatorización | Introducir ruido estadístico para distorsionar valores originales |
| Generalización | Reemplazar valores específicos por rangos o categorías más amplias |
| Seudonimización | Sustituir identificadores directos por seudónimos |

### Criterios de evaluación de efectividad

Equilibrio entre:
- **Utilidad de los datos**: preservar valor analítico.
- **Protección de privacidad**: minimizar riesgo de reidentificación.

### Herramientas de software libre

La guía destaca herramientas de software libre que facilitan el proceso de anonimización (detalle en documento PDF).

## 3. Casos de uso

Orientados a entidades y organizaciones que gestionan datos personales, cubriendo escenarios de publicación de información, investigación y análisis.

## 4. Marco normativo

La anonimización se posiciona como herramienta clave para cumplimiento de normativas vigentes de protección de datos personales y fomento de uso responsable de información.
