## Datos e Interoperabilidad

| Término | Dominio | Definición | Fuente |
| :--- | :--- | :--- | :--- |
| **API (Application Programming Interface)** | Datos/Interoperabilidad | Conjunto de definiciones y protocolos para comunicación estructurada entre aplicaciones; intermediario para intercambio de datos y servicios. | Estrategia de Datos de la Administración del Estado |
| **Dato** | Datos | Representación de atributo o variable (cuantitativa/cualitativa) capturada mediante observación o medición de hechos, personas o fenómenos. | Estrategia de Datos de la Administración del Estado |
| **Datos abiertos** | Datos/Transparencia | Datos digitales con características técnicas y jurídicas que permiten uso, reutilización y redistribución libre por cualquier persona u órgano estatal. | Estrategia de Datos de la Administración del Estado |
| **Datos maestros** | Datos/Gestión | Datos de contexto para actividad de negocio mediante conceptos comunes y abstractos (usuarios, productos, personal); referencia consistente en transacciones. | Estrategia de Datos de la Administración del Estado |
| **Datos de referencia** | Datos/Gestión | Conjuntos de datos que categorizan o relacionan otros datos mediante códigos; actúan como catálogos o listas controladas. | Estrategia de Datos de la Administración del Estado |
| **Integración de datos** | Datos/Infraestructura | Procesos de movimiento, consolidación y combinación coherente de datos entre almacenes y aplicaciones para análisis y toma de decisiones. | Estrategia de Datos de la Administración del Estado |
| **Interoperar** | Interoperabilidad | Comunicación o transferencia de datos en línea entre instituciones con mínima intervención humana, basada en estándares y protocolos comunes. | Estrategia de Datos de la Administración del Estado |
| **Metadatos** | Datos/Documentación | Datos que describen contexto, contenido y estructura de otros datos para su gestión, entendimiento y reutilización adecuada. | Estrategia de Datos de la Administración del Estado |
