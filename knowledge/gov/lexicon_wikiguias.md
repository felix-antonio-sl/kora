# Lexicón WikiGuías SGD — Definiciones Canónicas TDE

Fuente: WikiGuías SGD — `urn:gov:kb:lexicon-wikiguias`
Fuente primaria: Estrategia de Datos de la Administración del Estado (sección Glosario).
Uso: referencias desde otros artefactos TDE mediante XRef a `#ID`.

## Términos

| ID | Término | Dominio | Definición |
|---|---|---|---|
| TDE_DEF_API | API (Application Programming Interface) | Datos/Interoperabilidad | Conjunto de definiciones y protocolos que permiten a distintas aplicaciones o sistemas comunicarse entre sí de manera eficiente y estructurada, actuando como intermediarias para el intercambio de datos y servicios entre software dentro o entre plataformas |
| TDE_DEF_DATO | Dato | Datos | Representación de un atributo o variable cuantitativa o cualitativa mediante letras, números o símbolos, capturada a partir de la observación o medición de personas, objetos, hechos, juicios o fenómenos |
| TDE_DEF_DATOS_ABIERTOS | Datos abiertos | Datos/Transparencia | Datos digitales con características técnicas y jurídicas que permiten su uso, reutilización y redistribución libre por cualquier persona u órgano del Estado, promoviendo transparencia, responsabilidad y creación de valor mediante su publicación y reutilización |
| TDE_DEF_DATOS_MAESTROS | Datos maestros | Datos/Gestión | Datos que proporcionan contexto para los datos de actividad del negocio mediante conceptos comunes y abstractos (personas usuarias, productos, personal, dominios controlados), usados como referencia consistente en múltiples transacciones |
| TDE_DEF_DATOS_REFERENCIA | Datos de referencia | Datos/Gestión | Conjuntos de datos que categorizan o relacionan otros datos mediante códigos, sirviendo como catálogos o listas controladas que estructuran y clasifican datos operativos |
| TDE_DEF_INTEGRACION_DATOS | Integración de datos | Datos/Infraestructura | Procesos de movimiento, consolidación y combinación de datos de forma coherente dentro y entre almacenes de datos, aplicaciones y organizaciones, para disponer de información consistente y utilizable en análisis y toma de decisiones |
| TDE_DEF_INTEROPERAR | Interoperar | Interoperabilidad | Comunicar o transferir datos e información en línea entre instituciones con mínima intervención humana y basándose en estándares y protocolos comunes, de forma que los datos puedan ser utilizados de manera efectiva por las partes involucradas |
| TDE_DEF_METADATOS | Metadatos | Datos/Documentación | Datos que describen el contexto, el contenido y la estructura de otros datos, permitiendo entender, gestionar y reutilizar adecuadamente los conjuntos de datos |
| TDE_DEF_PLATAFORMA | Plataforma | Infraestructura Pública Digital | Software o conjunto de software, datos e infraestructura tecnológica que sustenta procesos o procedimientos, habilitando servicios y operaciones digitales transversales |
| TDE_DEF_SERVICIOS_COMPARTIDOS | Servicios compartidos | Infraestructura Pública Digital | Servicios o herramientas prestados por la Administración del Estado y utilizados por dos o más órganos o entidades, que consolidan, estandarizan y administran procesos y funciones comunes desde un centro de servicios |
| TDE_DEF_SERVICIOS_DIGITALES | Servicios digitales | Servicios Públicos Digitales | Prestaciones ofrecidas por los órganos de la Administración del Estado a sus usuarios a través de tecnologías digitales, que permiten realizar trámites y acceder a información y beneficios en línea |
| TDE_DEF_TECNOLOGIAS_DIGITALES | Tecnologías digitales | Tecnología/Infraestructura | Herramientas, sistemas automáticos, dispositivos, aplicaciones, recursos tecnológicos y de analítica de datos que generan, almacenan, procesan y gestionan información en entornos digitales |
