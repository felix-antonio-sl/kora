# RIS Genéricas — Programas de Inversión (SNI 2025)

SNI = Sistema Nacional de Inversiones. MDSF = Ministerio de Desarrollo Social y Familia. BIP = Banco Integrado de Proyectos. IDI = Iniciativa de Inversión.

Aplica: programas de inversión con duración definida y finita para población objetivo específica. Registro en BIP con código asignado. Evaluación: MDSF, nivel central o regional.

## Etapas

| Etapa | Alcance |
| :--- | :--- |
| 1. Perfil | Estudio preliminar; base obligatoria para postulación. |
| 2. Diseño | Planificación detallada de actividades, recursos y cronograma. |
| 3. Ejecución | Implementación física/financiera con presupuesto integral y seguimiento. |

## Antecedentes para Transición de Etapas

### Perfil → Diseño

1. Estudio de perfil completo según documento guía.
2. TDR (Términos de Referencia) específicos para etapa de diseño.
3. Presupuesto detallado exclusivo para la etapa de diseño.

### Perfil/Diseño → Ejecución

1. Estudio de perfil validado según metodología vigente.
2. Diseño definitivo del programa concluido.
3. Presupuesto detallado para la etapa de ejecución total.

## Contenido Mínimo del TDR

| Componente | Requisito |
| :--- | :--- |
| Diagnóstico | Situación base, problema central, caracterización de población objetivo. |
| Objetivos | Generales y específicos medibles. |
| Localización | Cobertura geográfica del estudio/programa. |
| Variables | Identificación, medición, análisis cuantitativo/cualitativo. |
| Actividades | Descripción pormenorizada de cada acción prevista. |
| Metodología | Técnicas aplicadas (talleres, e-learning, presenciales, etc.). |
| Cronograma | Carta Gantt en semanas o meses. |
| Resultados | Indicadores de producto y evaluación de impacto poblacional. |
| Informes | Cantidad, tipología, contenidos mínimos, hitos de entrega. |

## Presupuesto Detallado

### Etapa Diseño

- Asignaciones: "Consultorías" y "Gastos Administrativos" en ficha IDI.
- Personal: profesionales, técnicos, administrativos con perfiles detallados.
- Otros gastos: materiales, insumos, pasajes, difusión.
- Generales/utilidades incluidos.
- Unidad de medida: horas o unidades. Costo total incluye impuestos.

### Etapa Ejecución

- Incluir totalidad de gastos de la empresa ejecutora.
- Presupuesto anualizado obligatorio.
- "Consultorías" permitido para contraparte técnica.
- Separar gastos de licitación de los de seguimiento técnico.
- Unidad de medida: horas o unidades. Incluir impuestos.

### Normas de Clasificación

- Valores actualizados desglosados según clasificador presupuestario vigente.

## Registro en Ficha IDI

- Coherencia absoluta entre datos BIP y antecedentes técnicos (TDR).
- Estudios Básicos: registrar en "Gastos Administrativos" y "Consultoría y Contratación del Programa".
