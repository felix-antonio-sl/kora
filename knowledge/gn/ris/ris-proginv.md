# RIS Genéricas para Programas de Inversión (SNI 2025)

## Concepto General y Plataforma
- **Propósito:** Iniciativas de inversión con duración definida y finita para población objetivo específica.
- **Plataforma:** Registro en Sistema Nacional de Inversiones (BIP) con código asignado.
- **Evaluación:** Ministerio de Desarrollo Social y Familia (MDSF), niveles central o regional.

## Etapas del Programa de Inversión
| Etapa | Alcance y Requisito |
| :--- | :--- |
| 1. Perfil | Estudio preliminar; base obligatoria para postulación inicial. |
| 2. Diseño | Planificación detallada de actividades, recursos y cronograma. |
| 3. Ejecución | Implementación física/financiera con presupuesto integral y seguimiento. |

## Antecedentes para Transición de Etapas
### Transición Perfil a Diseño
- **Req-1:** Estudio de perfil completo según documento guía.
- **Req-2:** Términos de Referencia (TDR) específicos para etapa de diseño.
- **Req-3:** Presupuesto detallado exclusivo para la etapa de diseño.

### Transición Perfil/Diseño a Ejecución
- **Req-1:** Estudio de perfil validado según metodología vigente.
- **Req-2:** Diseño definitivo del programa concluido.
- **Req-3:** Presupuesto detallado para la etapa de ejecución total.

## Términos de Referencia (TDR) del Programa
| Componente | Requisitos Técnicos |
| :--- | :--- |
| Diagnóstico | Situación base, problema central y caracterización de población. |
| Objetivos | Definición de objetivos generales y específicos medibles. |
| Localización | Definición de cobertura geográfica del estudio/programa. |
| Variables | Identificación, medición y análisis cuantitativo/cualitativo. |
| Actividades | Descripción pormenorizada de cada acción prevista. |
| Metodología | Técnicas aplicadas (talleres, e-learning, presenciales, etc.). |
| Cronograma | Carta Gantt detallada por semanas o meses. |
| Resultados | Indicadores de producto y evaluación de impacto poblacional. |
| Informes | Cantidad, tipología, contenidos mínimos y hitos de entrega. |

## Presupuesto Detallado y Clasificador
### Requisitos por Etapa
- **Etapa Diseño:**
  - Asignaciones en "Consultorías" y "Gastos Administrativos".
  - Desglose Personal: Perfiles de profesionales, técnicos y administrativos.
  - Otros Gastos: Materiales, insumos, pasajes y difusión.
  - Generales/Utilidades: Incluir gastos generales y porcentaje de utilidad.
- **Etapa Ejecución:**
  - Incluir integridad de gastos de empresa ejecutora.
  - Presupuesto anualizado obligatorio.
  - Consultorías: Permitido para servicios de contraparte técnica.
  - Instrucción: Separar gastos de licitación de los de seguimiento técnico.

### Normas de Cálculo y Clasificación
- **Unidad de Medida:** Expresado en horas o unidades físicas.
- **Impuestos:** Costo total debe incluir IVA y gravámenes legales.
- **Clasificador:** Valores actualizados y desglosados según conceptos del clasificador presupuestario vigente.

## Registro en Ficha IDI
- **Coherencia:** Consistencia absoluta entre datos en plataforma BIP y antecedentes técnicos (TDR).
- **Asignaciones Específicas:** Estudios Básicos deben registrarse en "Gastos Administrativos" y "Consultoría y Contratación del Programa".
