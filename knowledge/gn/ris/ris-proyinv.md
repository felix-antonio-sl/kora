# RIS Genéricas — Proyectos de Inversión (SNI 2023)

Fuente: [PDF SNI](https://sni.gob.cl/storage/docs/RIS_genericas_para_proyectos_de_inversion_2023.pdf). SNI = Sistema Nacional de Inversiones. IDI = Iniciativa de Inversión. BIP = Banco Integrado de Proyectos. TDR = Términos de Referencia. UT = Unidad Técnica.

## Ciclo de Vida del Proyecto

| Estado | Etapas |
| :--- | :--- |
| Preinversión | Perfil → Prefactibilidad → Factibilidad |
| Inversión | Diseño → Ejecución |
| Operación | Puesta en marcha → Operación en régimen |

## Fase de Preinversión

- Perfil obligatorio al postular a prefactibilidad.
- Metodología: general o específica aplicable.
- Cronograma: Carta Gantt con fuentes de financiamiento y jornada (completa/parcial).
- Resultados: vinculados directamente a objetivos del estudio.
- Impacto ambiental: evaluación obligatoria.

### TDR para Estudios de Preinversión

| Ítem | Contenido requerido |
| :--- | :--- |
| Estudios previos | Nombre, consultora, año, resumen, versión más reciente. |
| Costo de inversión | Estimación inicial de montos. |
| Problema y objetivos | Definición del problema; objetivos generales y específicos. |
| Diagnóstico | Análisis de situación actual. |
| Variables | Oferta, demanda, proyecciones, brechas. |
| Alternativas de solución | Análisis comparativo. |
| Optimización | Tamaño óptimo, localización, momento de inversión. |
| Costos y beneficios | Identificación y valorización directos e indirectos. |
| Evaluación técnico-económica | Por cada alternativa. |
| Selección | Justificación de mejor alternativa; profundizar en factibilidad. |
| Resumen y conclusiones | Requerido. |

### Presupuesto en Preinversión

- Formato: desglose por asignaciones presupuestarias.
- Gastos administrativos: consultorías, personal (profesional, técnico, administrativo), gastos generales, utilidades.
- Consultorías: costo total, perfiles RRHH, unidades de medida (horas/unidades).
- Considerar: costos EIA/DIA si corresponde; montos incluyen impuestos; alineación con Clasificador Presupuestario y ficha IDI.

## Fase de Inversión

### Etapa de Diseño

| Documento | Requisito |
| :--- | :--- |
| Estudio preinversional | Perfil, prefactibilidad o factibilidad. |
| Programa arquitectónico | Según especificaciones sectoriales. |
| Planos de emplazamiento y zona | Áreas de influencia y servicios. |
| Carta Gantt | Duración en meses. |
| Calendario de inversión | Por asignaciones; incluye terreno si aplica. |
| TDR consultoría | Para especialistas: arquitectura, ingeniería, especialidades. |
| Presupuesto — consultorías | RRHH desagregado por nivel; materiales, insumos, difusión. |
| Presupuesto — revisores independientes | Según normativa vigente. |

Unidades: horas para RRHH, global para otros. Incluye impuestos.

### Etapa de Ejecución

| Documento | Requisito |
| :--- | :--- |
| Estudio preinversional | Actualizado con resultados de etapa de diseño. |
| Resultados de diseño | Planos, especificaciones técnicas, presupuestos visados. |
| Cotizaciones | Para equipos y equipamiento. |
| Carta Gantt | Detallada por actividad con fuentes de financiamiento. |

Asignaciones presupuestarias:

| Asignación | Contenido |
| :--- | :--- |
| Gastos administrativos | General. |
| Consultorías | General. |
| Terrenos | Incluye expropiación o adquisición. |
| Obras civiles | Incluye servidumbres y redes. |
| Equipamiento | Mobiliario integrado al proyecto. |
| Equipos | Máquinas, hardware, software. |
| Vehículos | Si forman parte integral del proyecto. |

Costo total incluye impuestos. Coherencia con ficha IDI y registros de respaldo.
