# RIS Proyectos Patrimoniales (SNI 2024)

Fuente: [PDF SNI](https://sni.gob.cl/storage/docs/RIS_Proyectos_Inmuebles_Patrimoniales_2024.pdf). Aplica: inversiones en inmuebles patrimoniales — monumentos históricos, paleontológicos, arqueológicos, inmuebles de conservación histórica, zonas de conservación.

Planes de infraestructura: generados por el Ministerio de las Culturas, las Artes y el Patrimonio. Propósito: poner en valor inmuebles y asegurar condiciones para bienes muebles protegidos.

## Consideraciones Transversales

- Riesgo de desastres (desde 2024): "Metodología Complementaria" para instalaciones turísticas.
- Enfoque de género: lineamientos específicos en la formulación de proyectos.

## Requisitos por Etapa

### Prefactibilidad y Factibilidad

| Documento | Requisito |
| :--- | :--- |
| Formulación del proyecto | "Metodología para la formulación y evaluación socioeconómica de proyectos de patrimonio cultural inmueble". Aplicar precios sociales. |
| Protección legal | Copia de Resolución o Decreto Oficial (CMN o MINVU) que avale la protección del inmueble. |
| Propiedad del terreno | Acreditar según "Requisitos genéricos sobre propiedades en que se proyectan edificaciones". |
| Presupuesto detallado | Consultorías, actividades (nombre, costo HH, insumos, gastos admin). Firmado por profesional responsable. |
| Aprobación de intervención | Documento de aprobación del proyecto por CMN o MINVU conforme a declaratoria original. |
| TDR consultoría | Profesionales especializados requeridos detallados. |
| Cronogramas | Carta Gantt de actividades + cronograma financiero. |

### Diseño

| Documento | Requisito |
| :--- | :--- |
| Actualización del perfil | Misma metodología sectorial integrando resultados de preinversión. |
| Protección legal | Ratificación mediante Resolución o Decreto Oficial. |
| Planos de emplazamiento | Con deslindes y cotas; concordancia con Carta Gantt y ficha IDI (Iniciativa de Inversión). |
| TDR diseño | Programa arquitectónico y modelo de gestión aprobado por UT (Unidad Técnica) y propietarios. |
| Presupuesto detallado | Consultorías, gastos admin, permisos vigentes, informes de revisores independientes si aplica. |
| Informes técnicos (opcional) | Estado de conservación, características especiales del terreno. |
| Carta Gantt | Licitación, adjudicación, contrato, control UT. |
| Cronograma financiero | Alineado con Carta Gantt y ficha IDI. |

### Ejecución

Modalidades de presentación:
- Directo a ejecución: perfil con metodología sectorial + diseño terminado.
- Desde diseño: resumen ejecutivo + diseño final.

| Documento | Requisito |
| :--- | :--- |
| Aprobación de diseño | Arquitectura aprobada por CMN o SEREMI MINVU (según corresponda). |
| Equipamiento | Lista detallada: ítems, cantidad, especificaciones técnicas, costo unitario y total (si aplica). |
| Presupuesto obras civiles | Ítems, cantidades, costos. |
| Presupuesto equipos/equipamiento | Separado, con respaldos. |
| Presupuesto consultorías/asesorías | Especializadas, si aplica. |
| Gastos administrativos | Detalle incluido. |
| Documentación de edificación | Entrega de documentos requeridos para proyectos constructivos generales. |
| TDR / especificaciones técnicas | Obras, experiencia exigida a oferentes, equipamiento y equipos. TDR específico para asesorías especializadas. |
| Carta Gantt | Integral: licitación, adjudicación, contrato, adquisiciones. |
| Cronograma financiero | Acorde a Gantt y ficha IDI. |
| Plan de contingencia | Si se retiran elementos patrimoniales: protocolo de almacenamiento y preservación durante ejecución. |
