# RIS Edificación Pública (SNI 2024)

Fuente: [PDF SNI](https://sni.gob.cl/storage/docs/RIS_-_Proyectos_de_Edificacion_publica_-_26-04-2024.pdf). Aplica: proyectos de edificación pública para ampliar capacidad y mejorar dependencias institucionales. Dudas técnicas: Jefatura División de Evaluación Social de Inversiones (MDSF).

Clasificación BIP (Banco Integrado de Proyectos): Sector Multisectorial / Subsector Administración Multisector / Descriptor Edificación Pública.

## Orientaciones Sectoriales

| Dimensión | Requisito |
| :--- | :--- |
| Arquitectura sustentable | Diseño integrado: climatización, iluminación, sanitario, aguas lluvias, estructuras, redes, ERNC, paisajismo. |
| Eficiencia de recursos | Uso eficiente de energía y agua; minimizar impacto ambiental y dependencia de hidrocarburos. |
| Accesibilidad universal | Libre acceso y circulación para personas con discapacidad. Normativa: Ley N°20.530 art. 13; DS N°47/1992 (OGUC); DS N°50/2015 (MINVU). |
| Evaluación técnica | Regional/Comunal: SEREMI Desarrollo Social; Nacional: Nivel Central MDSF. |

## Requisitos por Etapa

### Prefactibilidad y Factibilidad

- Metodologías: Edificación Pública + Riesgo de Desastre.
- Información base: estimaciones de fuentes secundarias.

### Diseño

| Requisito | Descripción |
| :--- | :--- |
| Metodologías | Edificación Pública + Riesgo de Desastre. |
| Proyección de dotación | Demanda proyectada de equipos, equipamiento y m² según misión del servicio y carga de trabajo. |
| Situación legal inmueble | Acreditación según "Requisitos Genéricos sobre las Propiedades en que se Proyectan Edificaciones". |
| Antecedentes técnicos | Según "Antecedentes Técnicos Requeridos para Proyectos que Consideran Edificación". |
| Compromiso de traslado | Carta por cada servicio involucrado, comprometiendo traslado al nuevo recinto. |
| Documento de administración | Firmado por jefes de servicios responsables de la infraestructura. |

### Ejecución

| Requisito | Descripción |
| :--- | :--- |
| Metodologías | Edificación Pública + Riesgo de Desastre (versión Ejecución). |
| Requisitos de diseño | Si no se postuló a Diseño en SNI: cumplir etapa diseño (especialmente cuando se concentran varios servicios). |
| Antecedentes técnicos | Según "Antecedentes Técnicos Requeridos para Proyectos que Consideran Edificación" (Ejecución). |
| Cuadro comparativo | Programa arquitectónico aprobado vs. diseño resultante, con justificación de diferencias. |
| Planificación | Cronograma de actividades o Carta Gantt. |

### Diseño y Ejecución Conjunta

| Requisito | Descripción |
| :--- | :--- |
| Formulación diseño | Requisitos de etapa de diseño con estimaciones de fuentes secundarias; metodologías correspondientes. |
| Normativa específica | "Instructivo para postulación de proyectos a etapa de ejecución con diseño referencial". |
| Eficiencia energética | Criterios de sostenibilidad según "Antecedentes Técnicos Requeridos para Proyectos que Consideran Edificación". |
